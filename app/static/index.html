Keyboard shortcuts

Existing viewer shortcuts grouped by purpose.

Navigation

  • A / D Previous / next section
  • PageUp / PageDown Previous / next section
  • ArrowLeft / ArrowRight Pan horizontally
  • R Reset to full view

Display

  • G Auto gain
  • Shift+G Reset gain
  • N Switch between the raw layer and the first computed layer

Picking

  • P Toggle pick mode
  • Ctrl+Z Undo manual pick action
  • Ctrl+Y / Ctrl+Shift+Z Redo manual pick action

Interaction

  • Alt Temporarily use pan interaction while held
  • ? Open shortcut help
  • Esc Close shortcut help

Section, display, and pick-mode shortcuts are active while the plot area is hovered. Shortcut help and manual pick undo/redo stay available across the viewer page unless focus is inside an editable control.
